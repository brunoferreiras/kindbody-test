# KindBody Test

## How to use

1. Start the ruby container with `make up`
```bash
make up
```

2. Enter the ruby container with `make bash`
```bash
make bash
```

3. Install the dependencies with `bundle install`
```bash
bundle install
```

4. Run the tests with `make test`

```bash
make test
```
